# 配信録画くん

配信サイトのライブストリーミングを簡単に録画できるGUIアプリケーションです。

## 概要

**配信録画くん**は、StreamlinkとPyQt6を使用して構築された配信動画の自動録画ツールです。複数の配信プラットフォームに対応しており、直感的なGUIで配信動画の録画を管理できます。

## 主な機能

- **複数プラットフォーム対応**: Streamlinkが対応する様々な配信サイトに対応
- **自動監視機能**: 指定したURLの配信状況を定期的に確認
- **自動録画**: 配信が開始されたら自動的に録画を開始
- **柔軟な設定**: 画質、再接続回数、待機間隔など細かい設定が可能
- **再接続機能**: 接続失敗時の自動再接続に対応

## 動作環境
- Windows

### 2. 必要なツール

https://www.ffmpeg.org/download.html
からWindows用のffmpegをダウンロードし、インストールしてください。インストール後、ffmpegのパスをシステム環境変数に追加してください。

### 基本的な操作

1. 配信URLを入力
2. 保存先ディレクトリを選択
3. 画質やその他の設定を調整
4. 「録画開始」ボタンをクリック

### 設定項目

- **画質**: best (推奨) / 1080p / 720p / 480p など
- **再接続回数**: 接続失敗時の最大再試行回数
- **再接続待機時間**: 再接続までの待機秒数
- **自動監視間隔**: 配信状況確認の間隔秒数

### 録画ができない場合

1. インターネット接続を確認してください
2. 配信URLが正しいことを確認してください
3. Streamlinkが指定の配信サイトに対応しているか確認してください

### ffmpegエラーが出た場合

ffmpegがインストールされていない可能性があります。上記の「インストール」セクションを参照してください。
