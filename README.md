# 配信録画くん

配信サイトのライブストリーミングを簡単に録画できるソフトウェアです。

## 対応プラットフォーム
- Twitch
- YouTube Live
- TikTok Live
- ツイキャス
- ニコニコ生放送

### 動作環境
- Windows

### 必要なツール

https://www.ffmpeg.org/download.html
からWindows用のffmpegをダウンロードし、インストールしてください。インストール後、ffmpegのパスをシステム環境変数に追加してください。

### 注意事項

YouTube Data APIの使用制限された場合、APIからAPI無しでの録画に自動的に切り替わります。
